#  Movie Recommendation System

A **content-based movie recommendation system** built using **Machine Learning in Python**.  
The system recommends movies based on **genre similarity** using **TF-IDF Vectorization** and **Cosine Similarity**.

---

##  Features
- Content-based movie recommendations
- Uses TF-IDF to convert movie genres into numerical features
- Computes similarity using cosine similarity
- Fast and lightweight
- Beginner-friendly ML project

---

##  Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity
